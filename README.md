# ICP Token Wallet

## Overview
Secure token wallet implementation on the Internet Computer Protocol.

## Features
- Token transfers
- Balance management
- Test addresses integration

## Setup Instructions
1. Install DFX:
```dfx start --clean --background```

2. Deploy:
```dfx deploy```

3. Run tests:
```cargo test```
